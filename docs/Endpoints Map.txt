/users:
	GET(): Return all users.
	POST(User): Create new User, return User or ErrorMessage.
	/user_id:
		GET(): Return user.
		Put(User): Update User, return User or ErrorMessage.
		Delete(): Delete user, retrun User or ErrorMessage.
		/role:
			GET(): Return user's role.
	
/activities:
	GET(): Return all activities.
	POST(Activity): Create activity, return Activity or ErrorMessage.
	/activity_id:
		GET(): Return activity.
		PUT(Activity): Update Activity, return Activity or ErrorMessage.
		DELETE(): Delete activity, retrun Activity or ErrorMessage.
