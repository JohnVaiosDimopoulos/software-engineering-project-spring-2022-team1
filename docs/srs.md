# Εισαγωγή

## Σκοπός
<p>
Σκοπός της εργασίας είναι να αναπτυχθεί μια διαδικτυακή πλατφόρμα, η οποία θα βοηθά τους
γονείς να επιλέγουν δραστηριότητες για τα παιδιά τους
</p>

## Εμβέλεια
- Ένας Γονέας μπορεί:
    - Nα ακυρώσει μία κράτηση εφόσον η αντίστοιχη δραστηριότητα δεν έχει ξεκινήσει ακόμη.
    - Nα αξιολογήσει έναν Πάροχο Υπηρεσιών ως προς μία δραστηριότητα, εάν έκανε κράτηση σε αυτή και δεν την ακύρωσε πριν την έναρξη της.
    - Nα κάνει κράτηση με την μορφή της συνδρομής, κάνοντας επαναληπτικά κράτηση σε διαφορετικές ημέρες για την ίδια δραστηριότητα ταυτόχρονα.
- Ένας Πάροχος Υπηρεσιών μπορεί να δηλώσει την περιοδικότητα μίας δραστηριότητας επιλέγοντας ταυτόχρονα όλες τις διαφορετικές ημέρες που διεξάγεται αυτή.
- Δεν υφίσταται κάποια αυτόματη χρέωση για τις περιοδικές δραστηριότητες. Ο γονέας μπορεί να κάνει κράτηση πολλαπλών sessions όποτε το επιθυμεί.
- To σύστημα προσομοιώνει την επικοινωνία με τραπεζικό σύστημα, κατά την αγορά πόντων από τον Γονέα και την εξαργύρωση πόντων του Παρόχου, χωρίς να υφίσταται πραγματικά.
- Για τις συναλλαγές στο σύστημα, υλοποιείται ένα σύστημα πόντων οι οποίοι ανταλλάσσονται μεταξύ Παρόχων Υπηρεσιών και Γονέων. Αναλυτικότερα:
    - Για την αγορά πόντων απαιτείται η μεταφορά χρημάτων, από κάρτα που έχει δηλώσει ένας Γονέας, στο λογαριασμός της εταιρίας.
        - Τα διαθέσιμα πακέτα είναι: 20 πόντοι με 10 ευρώ, 45 πόντοι με 20 ευρώ και 120 πόντοι με 50 ευρώ.
        -  Μέγιστη αναλογία πόντων προς ευρώ 2.4 πόντοι/ευρώ.
    - Για την εξαργύρωση πόντων μεταφέρεται το αντίστοιχο ποσό από τον λογαριασμό της εταιρίας, στον προηγουμένως δηλωμένο τραπεζικό λογαριασμό του Παρόχου Υπηρεσιών
        - H αναλογία εξαργύρωσης πόντων προς ευρώ είναι σταθερή 2.5 πόντοι/ευρώ
        - H εταιρία έχει επομένως κέρδος 0.1-0.5 πόντους/ευρώ ή 2-10 ευρώ/πόντο.
- Ένας διαχειριστής μπορεί να άλλάξει τον κωδικό, τον ρόλο και το όνομα ενός χρήστη, άμεσα χωρίς κάποια επιπλέον διαδικασία.
- Ένας διαχειριστής ενημερώνει ένα χρήστη για τις αλλαγές στον λογαριασμό του μέσω ξεχωριστής και μη εξαρτώμενης από το σύστημα μέσου/εφαρμογής.

## Γλωσσάρι  και Αναφορές
<p>
Στην έκταση του εγγράφου αναφέρονται οι εξής ρόλοι στην εφαρμογή
<ul>
    <li><b>Ανώνυμος χρήστης</b>: Κάθε χρήστης ο οποίος δεν έχει εγγραφεί επιτυχώς στο σύστημα</li>
    <li><b>Γονέας</b>: Χρήστης ο οποίος έχει εγγραφεί επιτυχώς στο σύστημα με σκοπό την αγορά δικαιώματος συμμετοχής σε δραστηριότητες για τα παιδιά τους.</li>
    <li><b>Πάροχος υπηρεσιών</b>: Χρήστης ο οποίος έχει εγγραφεί επιτυχώς στο σύστημα με σκοπό την πώληση δικαιωμάτων συμμετοχής σε δραστηριότητες σε <b>Γονείς</b> για τα παιδία τους.</li>
    <li><b>Διαχειριστής</b>: Χρήστης για τον οποίο έχει εξ αρχής δημιουργηθεί λογαριασμός στο σύστημα και διαχειρίζεται λογαριασμούς άλλων χρηστών.</li>
</ul>
</p>

# Περιγραφή

## Ανάγκες Χρηστών 
<p>
Στην έκταση του εγγράφου αναφέρονται οι εξής ρόλοι στην εφαρμογή
<ul>
    <li>Κάθε <b>Ανώνυμος χρήστης</b> θέλει να μπορεί να βλέπει τις δυνατότητες της εφαρμογής, χωρίς να δεσμευτεί να δημιουργήσει λογαριασμό, ώστε να εκτιμήσει αν δικαιολογούν τα προνόμια χρήσης της εφαρμογής, την περεταίρω ενασχόληση.</li>
    <li>Κάθε <b>Γονέας</b> θέλει να μπορεί να αναζητεί και να κάνει κράτηση αποτελεσματικά και εξατομικευμένα σε δραστηριότητες καθώς και να επιβλέπει και να διαχειρίζεται τις δραστηριότητες στις οποίες έχει κάνει ήδη κράτηση.</li>
    <li>Κάθε <b>Πάροχος υπηρεσιών</b> θέλει να δημοσιεύει αποτελεσματικά και εξατομικευμένα τις δραστηριότητες που παρέχει, καθώς και να επιβλέπει στοιχεία που περιέχουν πληροφορία για την επίδραση της εφαρμογής στην επιχειρησιακή του διαδικασία.</li>
    <li>Κάθε <b>Διαχειριστής</b> θέλει να επιβλέπει, δημιουργεί και τροποποιεί αποτελεσματικά και άμεσα λογαριασμούς άλλων χρηστών.</li>
</ul>
</p>

## Παραδοχές
<p>

<ul>
    <li>Κάθε δραστηριότητα διεξάγεται από έναν πάροχο.</li>
    <li>Κάθε δραστηριότητα γίνεται σε μία σαφώς καθορισμένη τοποθεσία/υποδομή.</li>
    <li>Κάθε δραστηριότητα έχει ένα πεπερασμένο πλήθος διαθέσιμων θέσεων.</li>
    <li>Κάθε τραπεζικός λογαριασμός που δίνεται από χρήστες είναι έγκυρος και με σωστά στοιχεία.</li>
    <li>Κάθε κάρτα που δίνεται από χρήστες είναι έγκυρη και με σωστά στοιχεία.</li> 
</ul>
</p>

# Απαιτήσεις
    
## Λειτουργικές Απαιτήσεις 

- Σαν Διαχειριστής, θέλω:
    - Να αναζητώ ένα χρήστη με βάση το username του.
    - Να αναζητώ ένα χρήστη με βάση το id του.
    - Να βλέπω τις αγορές ενός χρήστη.
    - Να εισάγω/δημιουργώ χρήστες στο σύστημα.
    - Να κάνω suspend χρήστες στο συστήματος.
    - Να κάνω reset τον κωδικό κάποιου χρήστη.
    - Να κάνω ορίζω/αλλάζω τον ρόλο κάποιου χρήστη.
    - Να συνδεθώ στον λογαριασμό μου.

- Σαν Ανώνυμος Χρήστης, θέλω:
    -  Να μπορώ να δω τις διαθέσιμες κατηγορίες δραστηριοτήτων.
    -  Να μπορώ να αναζητήσω δραστηριότητες με χρήση ελεύθερου κειμένου έτσι ώστε να δω τις “συγκεκριμένες”  δραστηριότητες.
    -  Να φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση την τιμή.
    -  Να φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση περιοχή.
    -  Να φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση ημερομηνία έναρξης.
    -  Να φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση ένα σύστημα αξιολόγησης.
    -  Να φιλτράρω τα αποτελέσματα μιας αναζήτησης βάση ηλικιακής κατηγορίας.
    -  Να φιλτράρω τα αποτελέσματα μιας αναζήτησης βάση κατηγορίας.
    -  Να ταξινομώ τα αποτελέσματα μιας αναζήτησης βάση τιμής.
    -  Να ταξινομώ τα αποτελέσματα μιας αναζήτησης βάση ημερομηνίας.
    -  Να ταξινομώ τα αποτελέσματα μιας αναζήτησης αλφαβητικά.
    -  Να ταξινομώ τα αποτελέσματα μιας αναζήτησης με βάση ένα σύστημα αξιολόγησης.
    -  Να δω όλες τις λεπτομέρειες μιας δραστηριότητας, σε ξεχωριστή σελίδα.
    -  Να μπορώ να δω στη σελίδα μιας δραστηριότητας άλλες δραστηριότητες που λαμβάνουν χώρα στο ίδιο Facility.

- Σαν Γονέας, θέλω:
    - Να μπορώ να επεξεργαστώ τα στοιχεία του λογαριασμού μου.
    - Να μπορώ να διαγράψω τον λογαριασμό μου.
    - Να  μπορώ να συνδεθώ στον λογαριασμό μου.
    - Να μπορώ να δημιουργήσω λογαριασμό.
    - Να μπορώ να κάνω “κριτική” σε μία δραστηριότητα που έχω αγοράσει αξιολογώντας με βάση ένα σύστημα 5 αστέρων.
    - Να φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση την ακτίνα από τη διεύθυνση που έχω δηλώσει.
    - Να μπορώ να βλέπω τις επερχόμενες δραστηριότητες που έχω ήδη κλείσει. 
    - Να έχω μου προτείνονται δραστηριότητες που είναι σχετικές με αυτές για τις οποίες έχω κάνει ήδη κράτηση.
    - Να έχω μου προτείνονται δραστηριότητες για τις οποίες έκανα κράτηση πρόσφατα.
    - Να μπορώ να αποθηκεύσω μία ή περισσότερες κάρτες μου για μελλοντική αγορά πόντων.
    - Να μπορώ να διαγράψω μία ή περισσότερες από τις κάρτες που έχω αποθηκεύσει.
    - Να μπορώ να ακυρώσω την κράτηση μου μιας δραστηριότητας και να μου επιστραφούν οι πόντοι, εφόσον δεν έχει ξεκινήσει.
    - Να μπορώ να δω το ιστορικό αγορών μου. 
    - Να μπορώ να αγοράσω μια δραστηριότητα εφόσον το κόστος της δεν υπερβαίνει τους διαθέσιμους πόντους μου.
    - Να μπορώ να κάνω κράτηση σε πολλαπλές ημερομηνίες για την ίδια δραστηριότητα ταυτόχρονα.
    - Να μπορώ να έχω και όλες τις δυνατότητες του ανώνυμου χρήστη.

- Σαν Πάροχος Υπηρεσιών, θέλω:
    - Να μπορώ να συνδεθώ στον λογαριασμό μου.
    - Να μπορώ να δημιουργήσω λογαριασμό.
    - Να μπορώ να επεξεργαστώ τα στοιχεία του λογαριασμού μου.
    - Να μπορώ να διαγράψω τον λογαριασμό μου.
    - Να δημοσιεύω μία δραστηριότητα που παρέχω.
    - Να δηλώσω τον τίτλο για μία δραστηριότητα που παρέχω.
    - Να δηλώσω τις ηλικίες για μία δραστηριότητα που παρέχω. 
    - Να δηλώσω την κατηγορία για μία δραστηριότητα που παρέχω.
    - Να δηλώσω την τιμή για μία δραστηριότητα που παρέχω.
    - Να δηλώσω τον αριθμό των διαθέσιμων θέσεων για μία δραστηριότητα που παρέχω..
    - Νπορώ να ανεβάσω μία φωτογραφία για μία δραστηριότητα που παρέχω. 
    - Να δηλώσω για μία δραστηριότητα που παρέχω αν είναι περιοδική ή όχι.
    - Να δηλώσω για μία δραστηριότητα που παρέχω και είναι περιοδική τις ημέρες και ώρες διεξαγωγής.
    - Να δηλώσω για μία δραστηριότητα που παρέχω και ΔΕΝ είναι περιοδική την ημέρα και ώρα διεξαγωγής.
    - Να δηλώσω για μία δραστηριότητα που παρέχω, ένα Facility, που προηγουμένως έχω δηλώσει, στο οποίο θα διεξαχθεί.
    - Να εφαρμόζεται υδατογράφημα (watermark) με μία λέξη/φράση που έχω δώσει σε κάθε φωτογραφία που ανεβάζω.
    - Ναι να δηλώνω τη λέξη/φράση που θα χρησιμοποιηθεί στα υδατογραφήματα.
    - Να βλέπω πόσες κρατήσεις έχουν γίνει για μία δραστηριότητα που παρέχω.
    - Να βλέπω το σύνολο των πόντων που μου έχουν αποφέρει οι κρατήσεις για μία δραστηριότητα.
    - Να βλέπω το σύνολο των χρημάτων που μου έχουν αποφέρει οι κρατήσεις για μία δραστηριότητα.
    - Να βλέπω το ποσοστό ακυρωμένων κρατήσεων για μία δραστηριότητα που παρέχω..
    - Να βλέπω τις κριτικές που έχουν κάνει οι χρήστες για μία δραστηριότητα μου
    - Να μπορώ να αποθηκεύσω έναν τραπεζικό λογαριασμό για μελλοντική εξαργύρωση πόντων.
    - Να μπορώ να αλλάξω τον τραπεζικό λογαριασμό που έχω αποθηκεύσει.
    - Να βλέπω το μέσο rating για μία δραστηριότητα που παρέχω..
    - Να επεξεργαστώ τα στοιχεία μιας δραστηριότητας που παρέχω.
    - Να δηλώνω ένα facility όπου έχω δικαίωμα να κάνω μία δραστηριότητα.
    - Να δηλώσω την διεύθυνση για ένα Facility.
    - Να δηλώσω την περιοχή για ένα Facility.
    - Να δηλώσω τον τίτλο για ένα Facility.
    - Να επεξεργαστώ τα στοιχεία για ένα Facility που έχω δηλώσει..
    - Να επεξεργαστώ τα στοιχεία για μία δραστηριότητα που έχω δηλώσει.
