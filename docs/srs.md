# Εισαγωγή

## Σκοπός
<p>
Σκοπός της εργασίας είναι να αναπτυχθεί μια διαδικτυακή πλατφόρμα, η οποία θα βοηθά τους
γονείς να επιλέγουν δραστηριότητες για τα παιδιά τους
</p>

## Εμβέλεια
<p>
Το σύστημα είναι ολοκληρωμένο από την αρχή μέχρι το τέλος και προϋποθέτει  σύστημα που... 
</p>

## Γλωσσάρι  και Αναφορές
<p>
Στην έκταση του εγγράφου αναφέρονται οι εξής ρόλοι στην εφαρμογή
<ul>
    <li><b>Ανώνυμος χρήστης</b>: Κάθε χρήστης ο οποίος δεν έχει εγγραφεί επιτυχώς στο σύστημα</li>
    <li><b>Γονέας</b>: Χρήστης ο οποίος έχει εγγραφεί επιτυχώς στο σύστημα με σκοπό την αγορά δικαιώματος συμμετοχής σε δραστηριότητες για τα παιδιά τους.</li>
    <li><b>Πάροχος υπηρεσιών</b>: Χρήστης ο οποίος έχει εγγραφεί επιτυχώς στο σύστημα με σκοπό την πώληση δικαιωμάτων συμμετοχής σε δραστηριότητες σε <b>Γονείς</b> για τα παιδία τους.</li>
    <li><b>Διαχειριστής</b>: Χρήστης για τον οποίο έχει εξ αρχής δημιουργηθεί λογαριασμός στο σύστημα και διαχειρίζεται λογαριασμούς άλλων χρηστών.</li>
</ul>
</p>

# Περιγραφή

## Ανάγκες Χρηστών 
<p>
Στην έκταση του εγγράφου αναφέρονται οι εξής ρόλοι στην εφαρμογή
<ul>
    <li>Κάθε <b>Ανώνυμος χρήστης</b> θέλει να μπορεί να βλέπει τις δυνατότητες της εφαρμογής, χωρίς να δεσμευτεί να δημιουργήσει λογαριασμό, ώστε να εκτιμήσει αν δικαιολογούν τα προνόμια χρήσης της εφαρμογής, την περεταίρω ενασχόληση.</li>
    <li>Κάθε <b>Γονέας</b> θέλει να μπορεί να αναζητεί και να κάνει κράτηση αποτελεσματικά και εξατομικευμένα σε δραστηριότητες καθώς και να επιβλέπει και να διαχειρίζεται τις δραστηριότητες στις οποίες έχει κάνει ήδη κράτηση.</li>
    <li>Κάθε <b>Πάροχος υπηρεσιών</b> θέλει να δημοσιεύει αποτελεσματικά και εξατομικευμένα τις δραστηριότητες που παρέχει, καθώς και να επιβλέπει στοιχεία που περιέχουν πληροφορία για την επίδραση της εφαρμογής στην επιχειρησιακή του διαδικασία.</li>
    <li>Κάθε <b>Διαχειριστής</b> θέλει να επιβλέπει, δημιουργεί και τροποποιεί αποτελεσματικά και άμεσα λογαριασμούς άλλων χρηστών.</li>
</ul>
</p>



## Παραδοχές
<p>

<ul>
    <li>Ένας Γονέας μπορεί να ακυρώσει μία κράτηση εφόσον η αντίστοιχη δραστηριότητα δεν έχει ξεκινήσει ακόμη</li>
    <li>Ένας Γονέας μπορεί να αξιολογήσει έναν Πάροχο Υπηρεσιών ως προς μία δραστηριότητα, εάν έκανε κράτηση σε αυτή και δεν την ακύρωσε πριν την έναρξη της.</li>
    <li>Ένας Γονέας μπορεί να κάνει κράτηση με την μορφή της συνδρομής, κάνοντας επαναληπτικά κράτηση σε διαφορετικές ημέρες για την ίδια δραστηριότητα ταυτόχρονα.</li>
    <li>Ένας διαχειριστής μπορεί να άλλάξει τον κωδικό, τον ρόλο και το όνομα ενός χρήστη, άμεσα χωρίς κάποια επιπλέον διαδικασία.</li>
    <li>Ένας διαχειριστής ενημερώνει ένα χρήστη για τις αλλαγές στον λογαριασμό του μέσω ξεχωριστής και μη εξαρτώμενης από το σύστημα μέσου/εφαρμογής.</li>
    <li>Ένας Πάροχος Υπηρεσιών μπορεί να δηλώσει την περιοδικότητα μίας δραστηριότητας επιλέγοντας ταυτόχρονα όλες τις διαφορετικές ημέρες που διεξάγεται αυτή.</li>
    <li>Για τις συναλλαγές στο σύστημα, υλοποιείται ένα σύστημα πόντων οι οποίοι ανταλλάσσονται μεταξύ Παρόχων Υπηρεσιών και Γονέων. Αναλυτικότερα:
        <ul>
            <li>Για την αγορά πόντων απαιτείται η μεταφορά χρημάτων, από κάρτα που έχει δηλώσει ένας Γονέας, στο λογαριασμός της εταιρίας.</li>
            <ul>
                <li>Τα διαθέσιμα πακέτα είναι: 20 πόντοι με 10 ευρώ, 45 πόντοι με 20 ευρώ και 120 πόντοι με 50 ευρώ.</li>
                <li> Μέγιστη αναλογία πόντων προς ευρώ 2.4 πόντοι/ευρώ.</li>
            </ul>
            <li> Για την εξαργύρωση πόντων μεταφέρεται το αντίστοιχο ποσό από τον λογαριασμό της εταιρίας, στον προηγουμένως δηλωμένο τραπεζικό λογαριασμό του Παρόχου Υπηρεσιών</li>
            <ul>
                <li>H αναλογία εξαργύρωσης πόντων προς ευρώ είναι σταθερή 2.5 πόντοι/ευρώ</li>
                <li>H εταιρία έχει επομένως κέρδος 0.1-0.5 πόντους/ευρώ ή 2-10 ευρώ/πόντο.</li>
            </ul>
        </ul> 
</ul>
</p>

# Απαιτήσεις
    
## Λειτουργικές Απαιτήσεις 

- Διαχειριστής:
    - Σαν Admin θέλω να αναζητώ ένα χρήστη με βάση το username του.
    - Σαν Admin θέλω να αναζητώ ένα χρήστη με βάση το id του.
    - Σαν Admin θέλω να βλέπω τις αγορές ενός χρήστη.
    - Σαν Admin θέλω να εισάγω/δημιουργώ χρήστες στο σύστημα.
    - Σαν Admin θέλω να κάνω suspend χρήστες στο συστήματος.
    - Σαν Admin θέλω να κάνω reset τον κωδικό κάποιου χρήστη.
    - Σαν Admin θέλω να κάνω ορίζω/αλλάζω τον ρόλο κάποιου χρήστη.
    - Σαν Admin θέλω να συνδεθώ στον λογαριασμό μου.

- Ανώνυμος Χρήστης:
    - Σαν Ανώνυμος Χρήστης θέλω να μπορώ να δω τις διαθέσιμες κατηγορίες δραστηριοτήτων.
    - Σαν Ανώνυμος Χρήστης θέλω να μπορώ να αναζητήσω δραστηριότητες με χρήση ελεύθερου κειμένου έτσι ώστε να δω τις “συγκεκριμένες”  δραστηριότητες.
    - Σαν Ανώνυμος Χρήστης φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση την τιμή.
    - Σαν Ανώνυμος Χρήστης φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση περιοχή.
    - Σαν Ανώνυμος Χρήστης φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση ημερομηνία έναρξης.
    - Σαν Ανώνυμος Χρήστης φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση ένα σύστημα αξιολόγησης.
    - Σαν Ανώνυμος Χρήστης φιλτράρω τα αποτελέσματα μιας αναζήτησης βάση ηλικιακής κατηγορίας.
    - Σαν Ανώνυμος Χρήστης φιλτράρω τα αποτελέσματα μιας αναζήτησης βάση κατηγορίας.
    - Σαν Ανώνυμος Χρήστης θέλω να ταξινομώ τα αποτελέσματα μιας αναζήτησης βάση τιμής.
    - Σαν Ανώνυμος Χρήστης θέλω να ταξινομώ τα αποτελέσματα μιας αναζήτησης βάση ημερομηνίας.
    - Σαν Ανώνυμος Χρήστης θέλω να ταξινομώ τα αποτελέσματα μιας αναζήτησης αλφαβητικά.
    - Σαν Ανώνυμος Χρήστης θέλω να ταξινομώ τα αποτελέσματα μιας αναζήτησης με βάση ένα σύστημα αξιολόγησης.
    - Σαν Ανώνυμος Χρήστης θέλω να δω όλες τις λεπτομέρειες μιας δραστηριότητας, σε ξεχωριστή σελίδα.
    - Σαν Ανώνυμος Χρήστης θέλω να μπορώ να δω στη σελίδα μιας δραστηριότητας άλλες δραστηριότητες που λαμβάνουν χώρα στο ίδιο Facility.

- Γονέας:
    - Σαν Γονέας θέλω να μπορώ να επεξεργαστώ τα στοιχεία του λογαριασμού μου.
    - Σαν Γονέας θέλω να μπορώ να διαγράψω τον λογαριασμό μου.
    - Σαν Γονέας θέλω να  μπορώ να συνδεθώ στον λογαριασμό μου.
    - Σαν Γονέας θέλω να μπορώ να δημιουργήσω λογαριασμό.
    - Σαν Γονέας θέλω να μπορώ να κάνω “κριτική” σε μία δραστηριότητα που έχω αγοράσει αξιολογώντας με βάση ένα σύστημα 5 αστέρων.
    - Σαν Γονέας θέλω να φιλτράρω τα αποτελέσματα μιας αναζήτησης με βάση την ακτίνα από τη διεύθυνση που έχω δηλώσει.
    - Σαν Γονέας θέλω να μπορώ να βλέπω τις επερχόμενες δραστηριότητες που έχω ήδη κλείσει. 
    - Σαν Γονέας θέλω να έχω μου προτείνονται δραστηριότητες που είναι σχετικές με αυτές για τις οποίες έχω κάνει ήδη κράτηση.
    - Σαν Γονέας θέλω να έχω μου προτείνονται δραστηριότητες για τις οποίες έκανα κράτηση πρόσφατα.
    - Σαν Γονέας θέλω να μπορώ να αποθηκεύσω μία ή περισσότερες κάρτες μου για μελλοντική αγορά πόντων.
    - Σαν Γονέας θέλω να μπορώ να διαγράψω μία ή περισσότερες από τις κάρτες που έχω αποθηκεύσει.
    - Σαν Γονέας θέλω να μπορώ να ακυρώσω την κράτηση μου μιας δραστηριότητας και να μου επιστραφούν οι πόντοι, εφόσον δεν έχει ξεκινήσει.
    - Σαν Γονέας θέλω να μπορώ να δω το ιστορικό αγορών μου. 
    - Σαν Γονέας θέλω να μπορώ να αγοράσω μια δραστηριότητα εφόσον το κόστος της δεν υπερβαίνει τους διαθέσιμους πόντους μου.
    - Σαν Γονέας θέλω να μπορώ να κάνω κράτηση σε πολλαπλές ημερομηνίες για την ίδια δραστηριότητα ταυτόχρονα.
    - Σαν Γονέας θέλω να μπορώ να έχω και όλες τις δυνατότητες του ανώνυμου χρήστη.

- Πάροχος Υπηρεσιών:
    - Σαν Πάροχος Υπηρεσιών θέλω να μπορώ να συνδεθώ στον λογαριασμό μου.
    - Σαν Πάροχος Υπηρεσιών θέλω να μπορώ να δημιουργήσω λογαριασμό.
    - Σαν Πάροχος Υπηρεσιών θέλω να μπορώ να επεξεργαστώ τα στοιχεία του λογαριασμού μου.
    - Σαν Πάροχος Υπηρεσιών θέλω να μπορώ να διαγράψω τον λογαριασμό μου.
    - Σαν Πάροχος Υπηρεσιών θέλω να δημοσιεύω μία δραστηριότητα που παρέχω.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω τον τίτλο για μία δραστηριότητα που παρέχω.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω τις ηλικίες για μία δραστηριότητα που παρέχω. 
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω την κατηγορία για μία δραστηριότητα που παρέχω.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω την τιμή για μία δραστηριότητα που παρέχω.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω τον αριθμό των διαθέσιμων θέσεων για μία δραστηριότητα που παρέχω..
    - Σαν Πάροχος Υπηρεσιών θέλω μπορώ να ανεβάσω μία φωτογραφία για μία δραστηριότητα που παρέχω. 
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω για μία δραστηριότητα που παρέχω αν είναι περιοδική ή όχι.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω για μία δραστηριότητα που παρέχω και είναι περιοδική τις ημέρες και ώρες διεξαγωγής.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω για μία δραστηριότητα που παρέχω και ΔΕΝ είναι περιοδική την ημέρα και ώρα διεξαγωγής.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω για μία δραστηριότητα που παρέχω, ένα Facility, που προηγουμένως έχω δηλώσει, στο οποίο θα διεξαχθεί.
    - Σαν Πάροχος Υπηρεσιών θέλω να εφαρμόζεται υδατογράφημα (watermark) με μία λέξη/φράση που έχω δώσει σε κάθε φωτογραφία που ανεβάζω.
    - Σαν Πάροχος Υπηρεσιών θέλω και να δηλώνω τη λέξη/φράση που θα χρησιμοποιηθεί στα υδατογραφήματα.
    - Σαν Πάροχος Υπηρεσιών θέλω να βλέπω πόσες κρατήσεις έχουν γίνει για μία δραστηριότητα που παρέχω.
    - Σαν Πάροχος Υπηρεσιών θέλω να βλέπω το σύνολο των πόντων που μου έχουν αποφέρει οι κρατήσεις για μία δραστηριότητα.
    - Σαν Πάροχος Υπηρεσιών θέλω να βλέπω το σύνολο των χρημάτων που μου έχουν αποφέρει οι κρατήσεις για μία δραστηριότητα.
    - Σαν Πάροχος Υπηρεσιών θέλω να βλέπω το ποσοστό ακυρωμένων κρατήσεων για μία δραστηριότητα που παρέχω..
    - Σαν Πάροχος Υπηρεσιών θέλω να βλέπω τις κριτικές που έχουν κάνει οι χρήστες για μία δραστηριότητα μου
    - Σαν Πάροχος Υπηρεσιών θέλω να μπορώ να αποθηκεύσω έναν τραπεζικό λογαριασμό για μελλοντική εξαργύρωση πόντων.
    - Σαν Πάροχος Υπηρεσιών θέλω να μπορώ να αλλάξω τον τραπεζικό λογαριασμό που έχω αποθηκεύσει.
    - Σαν Πάροχος Υπηρεσιών θέλω να βλέπω το μέσο rating για μία δραστηριότητα που παρέχω..
    - Σαν Πάροχος Υπηρεσιών θέλω να επεξεργαστώ τα στοιχεία μιας δραστηριότητας που παρέχω.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώνω ένα facility όπου έχω δικαίωμα να κάνω μία δραστηριότητα.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω την διεύθυνση για ένα Facility.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω την περιοχή για ένα Facility.
    - Σαν Πάροχος Υπηρεσιών θέλω να δηλώσω τον τίτλο για ένα Facility.
    - Σαν Πάροχος Υπηρεσιών θέλω να επεξεργαστώ τα στοιχεία για ένα Facility που έχω δηλώσει..
    - Σαν Πάροχος Υπηρεσιών θέλω να επεξεργαστώ τα στοιχεία για μία δραστηριότητα που έχω δηλώσει.
